# Sentiment Analysis of Amazon Reviews
This project performs Sentiment Analysis on Amazon reviews to classify sentiments as Positive, Neutral, or Negative. Using Natural Language Processing (NLP) techniques, we analyze customer feedback to understand user sentiment trends.

## 📌 Project Overview
## 📂 Dataset
- The dataset consists of Amazon product reviews.
- It includes text reviews, ratings, and other metadata.
- Data is preprocessed by cleaning text, removing stopwords, and tokenizing sentences.

## 🛠️ Technologies Used
- **Programming Language:** Python
- **Libraries:**
  - NLP: `nltk`, `textblob`, `vaderSentiment`
  - Data Handling: `pandas`, `numpy`, `re`
  - Visualization: `matplotlib`, `seaborn`, `wordcloud`, `plotly`, `cufflinks`


## 🚀 Features
- Preprocesses text data (removes stopwords, punctuations, tokenization)
- Uses **VADER Sentiment Analyzer** for review classification
- Provides **visualizations** (word clouds, sentiment distribution graphs)
- Supports **interactive plots** using Plotly

## 📊 Results & Analysis
- Displays sentiment breakdown with bar charts.
- Shows word frequency using WordClouds.
- Analyzes sentiment trends across different product categories.

## 🤖 Future Improvements
- Use deep learning models like BERT for sentiment classification.
- Expand dataset to multiple e-commerce platforms.
- Deploy as a web app using **Streamlit**.
